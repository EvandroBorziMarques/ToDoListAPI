# 	API de tarefas

## Features

- API CRUD que contém: :heavy_check_mark:
  - Tarefas
    - Criar tarefa;
    - Listar tarefas;
    - Buscar tarefa por ID;
    - Atualizar tarefa;
    - Deletar tarefa.
